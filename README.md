# Formulario
Sistema de Registro Seguro  Formulario web con enfoque en seguridad informática.  Características:  Seguridad: Cifrado de claves (BCRYPT) y consultas preparadas para evitar inyección SQL.  Validación: Doble verificación (JS y PHP) de formato, tipo y longitud.  Tecnologías: PHP, MySQL, JavaScript y CSS.
